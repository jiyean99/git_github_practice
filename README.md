## 프로젝트 개요

<img src="src/assets/img/gymkkong_img.png" alt="짐꽁" width="300" />  
피트니스 센터의 지점, 회원, 트레이너 정보를 통합 관리하는 **수업 예약·출석·결제 관리 시스템**입니다.  
회원·트레이너·관리자(지점 담당자)가 하나의 플랫폼에서 예약, 출석, 결제·환불, 커뮤니케이션을 편리하게 처리할 수 있도록 돕는 것을 목표로 합니다.

***

## 주요 기능 및 목표

- 피트니스 센터의 **지점·회원·트레이너 데이터를 한 번에 관리**할 수 있는 통합 플랫폼 구축  
- 예약, 출석, 결제·환불, 커뮤니케이션까지 **운영 전 과정을 온라인으로 전환**하여 업무 효율 상승  
- 실제 오프라인 센터 환경에서도 바로 사용할 수 있을 수준의 **실무 친화적인 시스템** 구현
